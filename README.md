# 🧠 RFP Writer Pro

AI-powered RFP proposal generator with 6 AI Agents + Second Brain

## ✨ Features

- 🧠 **Second Brain AI** - Neural network visualization during generation
- 🤖 **6 AI Agents** - Compliance, Writer, Pricing, Research, Quality, Legal
- 🌍 **25+ Languages** - Generate proposals in any language
- 💰 **Razorpay Payments** - Subscription and credit system
- 🔐 **Supabase Auth** - Email/Password + Google OAuth
- 📁 **File Upload** - PDF, DOCX, TXT parsing
- 📊 **Bid Tracker** - Track and manage bids
- 📚 **Template Library** - 50+ industry templates
